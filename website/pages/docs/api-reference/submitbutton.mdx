# SubmitButton

**\<SubmitButton />** is a simple button component for submitting the form. You can wrap a
SubmitButton inside the [\<FormBuilder>](/api-reference/formbuilder) component like so:

```tsx
import { FormBuilder, SubmitButton } from 'react-config-form';

const config = [...some config]

const handleSubmit = (form) => {...some submit handler}

<FormBuilder config ={config} onSubmit={handleSubmit}>
  /* this is the important bit */
  <SubmitButton text="text you wan't to display on the button" />
</FormBuilder>
```

### Props

Here's a list of all the props you can pass in to SubmitButton:

- <HighlightedText text="text" showBackticks />:
  <InlineCode text="string" /> - The button text. [By default, it'll show a 'Submit' text]
- <HighlightedText text="position" showBackticks />:
  <InlineCode text="left" /> or
  <InlineCode text="middle" /> or
  <InlineCode text="right" /> - The alignment for the submit button. [By default, it'll show up in the middle]
- <HighlightedText text="submitClass" showBackticks />:
  <InlineCode text="JSX Element" /> - A string that includes class(es) to be attached with the submit button.

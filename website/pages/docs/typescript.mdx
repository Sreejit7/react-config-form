# TypeScript

react-config-form comes with full [TypeScript](https://typescriptlang.org) support.

There are types for the config, the form state, the props etc. With the nice tools and
editor integrations, you'll get autocompletion and error checks directly in the editor.

You can import the types directly from the library and use in your forms, like so:

```typescript
import {
  FormBuilder,
  SubmitButton,
  /* here come the types */
  FormInputConfig,
  FormSubmitState,
} from 'react-config-form';

const config: FormInputConfig[] = [...];

const handleSubmit = (form: FormSubmitState) => {
  // ...do something with form
}

<FormBuilder
  config ={config}
  onSubmit={handleSubmit}
>
  <SubmitButton />
</FormBuilder>
```
